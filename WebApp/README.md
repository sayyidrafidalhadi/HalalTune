# HalalTune — Client

Premium Halal Audio streaming PWA.

## Deploy on Vercel
1. Push this folder to a GitHub repo
2. Import the repo on [vercel.com](https://vercel.com)
3. Framework Preset: **Other**
4. Root Directory: `/` (default)
5. Deploy ✓

## PWA Icons
- `appicon.png` — Android home screen icon
- `appleicon.png` — iOS "Add to Home Screen" icon
- `icon.png` — Browser favicon / general use
