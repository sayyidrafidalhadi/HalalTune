# HalalTune — Admin Center

Admin dashboard for managing tracks, users and analytics.

## Deploy on Vercel
1. Push this folder to a **separate** GitHub repo
2. Import the repo on [vercel.com](https://vercel.com)
3. Framework Preset: **Other**
4. Root Directory: `/` (default)
5. Deploy ✓

> Keep this repo **private** on GitHub.

## PWA
Add to home screen on any device for a native-feeling admin app.
The `appicon.png` file is used as the home screen icon on both Android and iOS.
