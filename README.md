# HalalTune

A monorepo for the HalalTune project, containing the Web App, Admin Dashboard, and Landing Page.

## Project Structure

- `Admin/`: Admin dashboard for management and moderation.
- `LandingPage/`: Marketing landing page for the application.
- `WebApp/`: The main client application (Premium Halal Audio streaming PWA).

## Development

Each subdirectory is a standalone project with its own dependencies. Navigate to a specific directory to start development:

### Web App
```bash
cd WebApp
npm install
npm run dev
```

### Admin Dashboard
```bash
cd Admin
npm install
npm run dev
```

### Landing Page
```bash
cd LandingPage
npm install
npm run dev
```
